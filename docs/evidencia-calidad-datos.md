#Calidad de Datos

A nivel de calidad de datos, se revisaron los siguientes apartados de las tablas:
*Nulos en columnas obligatorias, 
*Duplicados por PK
*Conteo de filas entre capas (raw vs staging )
Los resultados de estas validaciones, son exportadas a un Archivo Excel

Las evidencias de control generado, pueden ser validadas en el archivo quality_report_*.xlsx mas reciente
Tambien puede ejecutarse nuevamente el script validaciones.py para generar un nuevo archivo de reporte de calidad.
